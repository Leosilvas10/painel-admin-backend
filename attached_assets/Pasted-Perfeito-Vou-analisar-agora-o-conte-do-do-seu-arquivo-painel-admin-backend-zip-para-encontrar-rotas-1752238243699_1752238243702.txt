O que pode estar causando o erro então?
Se o erro ainda ocorre, existem 3 possibilidades principais:

1. Alguma rota está sendo montada dinamicamente (via variável)
Se em algum lugar do seu código você monta uma rota com concatenação de string e esquece o nome do parâmetro, isso pode gerar o erro, mas não aparece como hard-coded nos arquivos.

Exemplo de erro:

Verifique se existe código que monta rotas a partir de variáveis ou strings.

2. Configuração incorreta de dependências
Alguma dependência está passando um path inválido para o path-to-regexp.

Isso pode vir de algum plugin, middleware customizado, ou configuração automática de rotas.

Se você tem algum CRUD automático, scaffold, plugin, etc., revise a configuração.

3. Algum arquivo ou rota foi alterado/recentemente adicionado
Se você editou recentemente algum arquivo de rota, pode ser que o erro ficou "escondido" num ponto não capturado pela busca, ou não foi salvo corretamente.
