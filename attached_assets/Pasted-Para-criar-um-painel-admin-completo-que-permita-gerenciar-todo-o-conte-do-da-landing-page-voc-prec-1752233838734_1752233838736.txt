Para criar um painel admin completo que permita gerenciar todo o conteúdo da landing page, você precisará das seguintes rotas de backend:

1. Autenticação
POST /api/auth/login - Login do administrador
POST /api/auth/logout - Logout do administrador
GET /api/auth/me - Verificar se está autenticado
2. Gerenciamento de Logo
POST /api/logo/upload - Upload da nova logo
GET /api/logo - Obter logo atual
DELETE /api/logo - Excluir logo atual
3. Gerenciamento de Vídeos
POST /api/videos/upload - Upload de vídeo
GET /api/videos - Listar todos os vídeos
GET /api/videos/:id - Obter vídeo específico
DELETE /api/videos/:id - Excluir vídeo
PUT /api/videos/:id - Atualizar metadados do vídeo
4. Gerenciamento de Conteúdo Textual
GET /api/content/sections - Listar todas as seções
GET /api/content/sections/:section - Obter conteúdo de seção específica
PUT /api/content/sections/:section - Atualizar conteúdo de seção
5. Gerenciamento de Blocos de Conteúdo
GET /api/blocks - Listar todos os blocos
POST /api/blocks - Criar novo bloco
GET /api/blocks/:id - Obter bloco específico
PUT /api/blocks/:id - Atualizar bloco
DELETE /api/blocks/:id - Excluir bloco
PUT /api/blocks/reorder - Reordenar blocos
6. Configurações Gerais
GET /api/settings - Obter todas as configurações
PUT /api/settings - Atualizar configurações gerais
PUT /api/settings/seo - Atualizar configurações de SEO
7. Upload de Imagens Gerais
POST /api/images/upload - Upload de imagens
GET /api/images - Listar imagens
DELETE /api/images/:id - Excluir imagem
Exemplo de Estrutura de Dados:
Configurações Gerais:

{
  "title": "Consórcio Imobiliário Banco Jota",
  "description": "Casa própria sem juros",
  "whatsapp": "5511999999999",
  "email": "contato@bancojota.com.br",
  "metaTitle": "SEO Title",
  "metaDescription": "SEO Description"
}
Bloco de Conteúdo:

{
  "id": "hero-section",
  "type": "hero",
  "order": 1,
  "content": {
    "title": "Realize o Sonho da Casa Própria",
    "subtitle": "Sem Juros",
    "description": "Consórcio imobiliário...",
    "image": "/uploads/hero-bg.jpg",
    "video": "/uploads/video-123.mp4"
  },
  "enabled": true
}
Essas rotas permitirão um controle completo sobre todo o conteúdo da landing page através do painel administrativo, mantendo a flexibilidade para futuras expansões.
